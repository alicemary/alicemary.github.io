Design Philosophy

Website Design

Progressive Abacus was created as a user-friendly interface for generating abacus worksheets. It includes a question generator built with JavaScript to restrict questions based on technique and difficulty. The site was constructed using the Middleman static site generator along with HTML and SCSS. Images and icons are original and required Adobe Illustrator and Photoshop.
